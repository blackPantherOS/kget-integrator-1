# KGet Integrator


Integrate [KGet](https://www.kde.org/applications/internet/kget/) Download Manager with Google Chrome, Chromium.

# Installation
run install.sh
```bash
sh install.sh
```

Open Chrome/Chromium -> Extension -> then drag and drop extension.crx to chrome
